# Remote Calculator

## Files:
- `server.py` - Calculator server
- `client.py` - Calculator client

## How to Run:

1. Start server: `python server.py`
2. Run client: `python client.py`
3. Enter first number, operator (+, -, *, /, %), second number
4. View result

## Operations Supported:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
